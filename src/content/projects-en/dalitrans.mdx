# Dalitrans s.r.o.

## Website for Freight and Logistics Company.

### Client

Dalitrans s.r.o.

### The project was worked on by

Gabriel & Marcel & Freelance Graphic Designer

### Link to the project

www.dalitrans.sk

### Description of the project

Dalitrans company has decided to develop a one-page website with various animations to provide a simple informational web presence for their freight and logistics services.

After the design was approved, we embarked on the project implementation with the goal of creating a modern and attractive online website. The website provides customers with an overview of Dalitrans' services and offerings, including transportation options and logistics solutions. Additionally, we implemented animations to add interactivity and appeal to website visitors.

The overall objective of the project was to create a simple, clear, and engaging informational website that effectively communicates key information about the services and logistics provided by Dalitrans.
